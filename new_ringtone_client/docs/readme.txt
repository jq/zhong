配置new_ringtone_client

1)检出代码，按如下顺序checkout
  1.1 util
  1.2 ring
  1.3 facebook_android
  1.4 new_ringtone_client
	
2)创建目录变量
	Window->Preference->General->Workspace->Linked Resources，点击'new' button
	
	新建三个Variable：
	2.1 facebook_src，指向你本地的facebook项目src的绝对路径
	例如，facebook_src = "D:\workspace\facebook_android\src"
	2.2 ring_src，指向你本地的ring项目的src绝对路径
      例如，ring_src = "D:\workspace\ring\src"
	2.3 util_src，指向你本地的util项目的src绝对路径
      例如， util_src = "D:\workspace\util\src" 或者， util_src = "C:\workspace\util\src"
